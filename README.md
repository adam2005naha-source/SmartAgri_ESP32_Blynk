# Smart Agriculture ESP32 + Blynk

## Description
Surveillance de l'humidité du sol et de l'air avec ESP32 et DHT11, contrôle automatique d'une pompe via Blynk.

## Composants
- ESP32
- DHT11
- Capteur d'humidité du sol
- Relais pour pompe
- Application Blynk

## Installation
1. Installer Arduino IDE.
2. Installer les bibliothèques `Blynk` et `DHT`.
3. Modifier le code `.ino` avec ton `auth token` Blynk et tes infos WiFi.
4. Charger le code sur ESP32.
5. Connecter le projet sur Blynk App.
